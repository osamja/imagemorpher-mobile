Users must upload two pictures for the morpher to function, the pictures uploaded explicitly by the user for the morpher are collected.

The uploaded images are being collected because they serve as the input for the morpher.  The morpher takes in two images (faces uploaded by the user) and returns the morphed/blended picture as the output .  The image collection is necessary for core app functionality, there are no other planned uses of this data.

This data will not be shared with any third parties, and the information will be stored on the backend server where the API call is being made (https://sammyjaved.com)
